# Counter Douglas Global Offensive

[![CI](https://github.com/beanthemoonman/dougiesbigtrip/actions/workflows/ci.yml/badge.svg)](https://github.com/beanthemoonman/dougiesbigtrip/actions/workflows/ci.yml)

A browser FPS demo that imitates the look and feel of **Counter-Strike: Source** using only CC0/permissively-licensed assets.

## Getting Started

```bash
pnpm install
pnpm dev
```

## Commands

| Command | Description |
|---|---|
| `pnpm dev` | Vite dev server |
| `pnpm build` | Production build |
| `pnpm typecheck` | TypeScript type checking |
| `pnpm test` | Vitest test suite |
| `pnpm lint` | ESLint |
| `pnpm format` | Prettier |
